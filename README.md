# Website Portofolio Personal - Green Theme

Website ini adalah portofolio personal sederhana yang dibangun menggunakan **HTML**, **CSS**, dan **JavaScript**. Desain website mengusung tema Hijau dan Putih yang terinspirasi dari identitas visual UIN Alauddin Makassar.

## Fitur Utama
1.  **Responsif:** Tampilan menyesuaikan perangkat (Desktop, Tablet, Mobile).
2.  **Hamburger Menu:** Navigasi interaktif untuk tampilan mobile.
3.  **Tabel Data Diri:** Menampilkan biodata dalam format tabel yang rapi.
4.  **Hero Section:** Tampilan awal yang menarik dengan tombol interaktif.

## Struktur File
* `index.html`: Struktur utama halaman website.
* `style.css`: File gaya eksternal untuk desain (warna, layout, responsivitas).
* `script.js`: File logika eksternal untuk interaktivitas (menu mobile, tombol).

## Cara Menjalankan
1.  Unduh atau clone folder proyek ini.
2.  Pastikan semua file (`index.html`, `style.css`, `script.js`) berada dalam satu folder yang sama.
3.  Buka file `index.html` menggunakan browser web (Chrome, Firefox, Edge, dll).

## Penjelasan Kode
* **CSS:** Menggunakan CSS Variables (`:root`) untuk manajemen warna hijau (`#006837`) agar mudah diubah. Menggunakan Flexbox untuk Navigasi dan Grid untuk Portofolio.
* **JS:** Menggunakan `document.querySelector` untuk memilih elemen DOM hamburger menu dan memberikan event `click` untuk menampilkan/menyembunyikan navigasi pada layar kecil.
