Website ini merupakan portofolio personal sederhana yang dibangun menggunakan **HTML**, **CSS**, dan **JavaScript**.  
Desain website mengusung tema **Hijau dan Putih** yang terinspirasi dari identitas visual **UIN Alauddin Makassar**.

Website ini dibuat sebagai media latihan untuk memahami dasar-dasar pengembangan web statis.

---

## Fitur Utama
1. **Responsif**  
   Tampilan website menyesuaikan berbagai ukuran layar, mulai dari desktop, tablet, hingga perangkat mobile.

2. **Hamburger Menu**  
   Menu navigasi interaktif yang muncul pada tampilan mobile untuk meningkatkan kenyamanan pengguna.

3. **Tabel Data Diri**  
   Informasi biodata ditampilkan dalam bentuk tabel agar terlihat rapi dan mudah dibaca.

4. **Hero Section**  
   Bagian tampilan awal (landing page) yang menarik dan dilengkapi dengan tombol interaktif.

---

## Struktur File
- `index.html`  
  Berisi struktur utama halaman website.

- `style.css`  
  Berisi pengaturan tampilan website seperti warna, layout, dan responsivitas.

- `script.js`  
  Berisi kode JavaScript untuk menangani interaksi pengguna, seperti menu mobile dan tombol.

---

## Cara Menjalankan Website
1. Unduh atau clone folder proyek ini.
2. Pastikan seluruh file (`index.html`, `style.css`, dan `script.js`) berada dalam satu folder.
3. Buka file `index.html` menggunakan browser web seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge.

---

## Penjelasan Kode
- **CSS**  
  Menggunakan *CSS Variables* (`:root`) untuk mengatur warna utama hijau (`#006837`) sehingga mudah diubah.  
  Layout menggunakan **Flexbox** untuk navigasi dan **CSS Grid** untuk bagian portofolio/kontak.

- **JavaScript**  
  Menggunakan `document.querySelector()` untuk memilih elemen DOM seperti hamburger menu, kemudian menambahkan *event listener* `click` untuk menampilkan atau menyembunyikan navigasi pada layar kecil.
